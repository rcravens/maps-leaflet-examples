US States

Map showing state boundarys that are imported from geoJson data. The map also shows a legend and an active info box. map

US Counties

Map showing all US counties that are imported from a shape file. The map is colored by land area (km2). map

US Cities

Map showing the largest 1000 US cities that are imported from a JSON file. The cities are evenly divided into "clinics" and "competitors". The map also incorporates a layers control that allows you to change the map type and show/hide clinics and competitors. map
